# allproperty

> allproperty mini admin panel

## Build Setup

### Required node.js!

``` bash
# install dependencies
$ npm install
```

open nuxt.config.js and change this url if you have something else

``` bash
axios: {
        baseURL: 'http://allproperty.local/api'
    },
```

``` bash
# serve with hot reload at localhost:3000
$ npm run dev
```
